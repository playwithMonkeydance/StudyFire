# 🚀 StudyFire AI - Smart AI-Powered Student Learning Platform

> A modern, gamified, and AI-powered study assistant built to help students learn smarter, stay consistent, and improve productivity through an engaging experience.

---

## 📌 Overview

StudyFire AI is a web-based intelligent learning platform designed for students.  
It combines **AI assistance + gamification + productivity tools** to create a daily study habit and improve learning efficiency.

This project is built using **HTML, CSS, and JavaScript**, with a focus on clean UI, responsiveness, and future AI integration.

---

## ✨ Key Features

### 🧠 AI Study Assistant
- Ask any academic question
- Step-by-step explanations
- Simplified “Explain Like I’m 10” mode
- Multi-subject support (Maths, Science, English, ICT)

---

### 🔥 Gamification System
- Daily streak tracking system
- XP (Experience Points) for every activity
- Level progression system
- Achievement badges & rewards

---

### 🏆 Leaderboard System
- Global ranking based on XP
- Weekly & monthly leaderboards
- Competitive learning environment

---

### ⏱️ Productivity Tools
- Pomodoro Timer for focused study sessions
- Quick notes system
- Flashcard-style revision support

---

### 🔔 Smart Notifications
- Daily study reminders
- Streak loss alerts
- Exam countdown notifications

---

## 🛠️ Tech Stack

- HTML5 → Structure
- CSS3 → Styling & UI design
- JavaScript (Vanilla) → Logic & interactivity
